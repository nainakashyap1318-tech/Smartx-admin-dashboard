# Admin Dashboard UI

A responsive admin dashboard UI built using HTML, CSS, and JavaScript.  
This project focuses on layout structuring, sidebar navigation, responsive behavior, and data visualization.

## Overview
This dashboard UI represents a typical admin panel used in real-world applications like analytics dashboards, admin systems, and management panels.  
It includes a collapsible sidebar, responsive layout, cards, and charts.

## Features
- Responsive dashboard layout (mobile-first)
- Sidebar with hamburger toggle
- Navigation links (Dashboard, Users, Products, Analytics, Settings)
- Dashboard cards for quick stats
- Chart integration for data visualization
- Clean and modern UI design

## Tech Stack
- HTML5  
- CSS3 (Flexbox & Media Queries)  
- JavaScript (DOM manipulation)
- Chart.js (for charts)
